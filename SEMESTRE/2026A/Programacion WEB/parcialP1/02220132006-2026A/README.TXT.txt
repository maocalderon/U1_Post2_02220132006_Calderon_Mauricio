Se implementaron las siguientes funcionalidades adicionales:
1. Sistema de cambio de tema (Modo Claro/Oscuro) mediante clases de CSS

2. Un contador dinamico de tarjetas que se actualiza en tiempo real

   cantidad de elementos visibles.

   un interruptor (toggle) para mostrar el ahorro en el pago anual.
4. Saludo personalizado en el Hero que detecta la hora del sistema 
   para dar la bienvenida (Buenos dias/tardes/noches).
